# Carsonella — Roadmap

Куда идём по физике и геймплею. Детали реализации и лог изменений — в [`README.md`](README.md).

Маркеры: ✅ готово · 🟡 частично · ⬜ ещё нет.

## Физика — ещё нет

- ⬜ **BBN — первичный нуклеосинтез.** Главный природный источник ⁷Li во Вселенной (не звёзды). Требует понятия «эпох» симуляции.

- ⬜ **Spallation (космические лучи).** Единственный путь к ⁹Be, ¹⁰B, ¹¹B и добавочному ⁶Li — α-цепочка перепрыгивает дыры A=5 и A=8. Включать когда в космосе уже накоплены C/N/O.

- 🟡 **Нечётные-Z элементы выше N.** Al, Cl, K и далее — нужны s-/r-процессы (захват нейтронов в AGB-звёздах и сверхновых) или CNO-продолжения. **(n,γ) захват подключён** через generic поле `neutronGammaResult` + правило `StarNeutronGammaReaction`; 11 реакций на имеющихся target-ядрах (см. ниже). β⁻-распады нейтрон-избыточных изотопов и target-ядра выше ⁵⁶Ni ещё впереди.

- 🟡 **Каталитическое горение водорода через (p,γ)/(p,α).** Семейство «4p → ⁴He с катализатором». Generic-инфраструктура: поля `protonGammaResult`/`protonAlphaResult` в Details + объединённое правило `StarProtonCaptureReaction` (roulette-wheel branching, rate/branching ratios захардкожены по target-ядру).
  - ✅ **Cold CNO-I/II/III/IV** — branching ~10%/90% на ¹⁵N/¹⁷O/¹⁸O, bottleneck ¹⁴N(p,γ) `rate=0.02`, slowdown ¹⁶O(p,γ) `rate=0.1`.
  - ✅ **NeNa-cycle** — ²⁰Ne→²¹Na→²¹Ne→²²Na→²²Ne→²³Na→²⁰Ne+α. β⁺-распады ²¹Na, ²²Na — generic `BetaPlusDecay`.
  - ✅ **MgAl-cycle** — ²⁴Mg→²⁵Al→²⁵Mg→²⁶Al→²⁶Mg→²⁷Al→²⁴Mg+α. β⁺-распады ²⁵Al, ²⁶Al — generic `BetaPlusDecay`.
  - ✅ **Inter-cycle leaks** — ²³Na+p→²⁴Mg+γ (NeNa→MgAl мост, branching 0.01%) и ²⁷Al+p→²⁸Si+γ (MgAl выход в Si, 1%).
  - 🟡 **(p,n) канал** — `Outcome.Neutron` + поле `protonNeutronResult` + третий вес в `branchingWeights`. Подключена ⁷Li(p,n)⁷Be (Q=-1.64 МэВ) — единственный target пока, solo-канал вне CNO. На CNO/NeNa/MgAl target-ядрах (¹³C/¹⁵N/¹⁷O/¹⁸O/²¹Ne/²²Ne/²⁵Mg/²⁶Mg) (p,n) реакции физически существуют, но эндотермичны и потребуют тонкой настройки branching ratios.
  - ⬜ **Hot CNO-I/II/III** — при высокой T протон-захваты обгоняют β⁺-распады. Требует горячую температурную моду.

- 🟡 **α-индуцированные реакции.** Каналы (α,γ), (α,p), (α,n), (α,α). Сейчас работает первый, остальные — по мере появления target-ядер.
  - ✅ **(α,γ)** — главная α-цепочка ⁴He→⁸Be→…→⁵⁶Ni плюс боковые ветки в hot CNO, AGB, α-rich freeze-out. Конкретные реакции — поля `alphaGammaResult` в `Entity.kt`.
  - 🟡 **(α,p)** — реализованы все экзотермические каналы на текущих target-ядрах в `Space`-моде (наш аналог «лаборатории»): историческая ¹⁴N(α,p)¹⁷O + ¹³N→¹⁶O, ¹⁵N→¹⁸O, ¹⁷F→²⁰Ne (HCNO breakout), ²⁸Si→³¹P (Si-burning). В `Star`-моде пока выключено — включим с появлением `TemperatureMode.HotStar`. Расширение ждёт ²²Ne, ²⁶Mg, Al, Cl.
  - 🟡 **(α,n)** — главный нейтронный источник для s-процесса. Реализованы семь каналов в `Star`-моде через generic `alphaNeutronResult`: ¹³C(α,n)¹⁶O (главный «main» s-process в AGB-звёздах), ¹⁷O(α,n)²⁰Ne, ¹⁸O(α,n)²¹Ne, ²¹Ne(α,n)²⁴Mg, ²²Ne(α,n)²⁵Mg (главный «weak» s-process в массивных звёздах), ²⁵Mg(α,n)²⁸Si, ²⁶Mg(α,n)²⁹Si. На ¹³C, ¹⁸O, ²²Ne, ²⁶Mg конкурирует с (α,γ) на тех же target-ядрах — выбор канала случайный (`weight()=0f`), оба наблюдаемы. Ждут добавления как target-ядра: ⁹Be, ¹⁰B, ¹¹B (источники в spallation-обогащённой среде).

- 🟡 **n-индуцированные реакции — путь к настоящему s-процессу.** Каналы (n,γ), (n,p), (n,α). Свободные нейтроны рождаются из (α,n), C/O-burning и (p,n).
  - 🟡 **(n,γ)** — захват нейтрона ядром, основа s-процесса. Нет кулоновского барьера. Generic поле `neutronGammaResult` + правило `StarNeutronGammaReaction`. Подключены 12 каналов: ¹H(n,γ)²H (BBN), ¹²C(n,γ)¹³C (замыкает цикл воспроизводства нейтронов через ¹³C(α,n)¹⁶O), ¹⁴N→¹⁵N, ¹⁶O→¹⁷O, ¹⁷O→¹⁸O, ²⁰Ne→²¹Ne, ²¹Ne→²²Ne, ²⁴Mg→²⁵Mg, ²⁵Mg→²⁶Mg, ²⁸Si→²⁹Si, ²⁹Si→³⁰Si, ³⁰Si→³¹Si (³¹Si β⁻-нестабилен → толкает цепочку на P, см. β⁻-распад ниже).
  - 🟡 **Target-ядра выше ⁵⁶Ni** — классический s-process путь через Fe-peak: ⁵⁶Fe→⁵⁷Fe→⁵⁸Fe→⁵⁹Fe(β⁻)⁵⁹Co→⁶⁰Co(β⁻)⁶⁰Ni→…→²⁰⁸Pb; именно через него во вселенной появляются ~50% ядер тяжелее железа. **Начато** (всё в текущем `Star`-режиме — s-процессу не нужен HotStar: нейтронный захват без кулоновского барьера, «медленность» = захват реже β⁻): ⁵⁶Fe→⁵⁷Fe→⁵⁸Fe→⁵⁹Fe(β⁻)⁵⁹Co→⁶⁰Co(β⁻)⁶⁰Ni→⁶¹Ni→⁶²Ni→⁶³Ni(β⁻)⁶³Cu (непрерывная (n,γ)/β⁻ цепочка от железа до меди). Добавлены изотопы ⁵⁷/⁵⁸/⁵⁹Fe, ⁶⁰Co, ⁶¹/⁶²/⁶³Ni и **три зажелезных элемента**: кобальт, никель s-путём (⁶⁰Ni — не α-цепочечный ⁵⁶Ni) и медь (⁶³Cu — первый новый Z=29). Полные ионизационные цепочки; energyLevels изотопа берутся от соседа того же Z, а для нового Z (Cu) — водородоподобные + реальная Iₚ нейтрали (приближение). Дальше по порядку: ⁶³Cu(n,γ)⁶⁴Cu(β⁻)⁶⁴Zn → … десятки изотопов до Pb.
  - 🟡 **β⁻-распад** — нейтрон-избыточные продукты (n,γ) распадаются с Z→Z+1, толкая s-process цепочку вверх по таблице. Generic поле `betaMinusDecayResult` + правило `BetaMinusDecay` (зеркало `BetaPlusDecay`) **подключены**. Targets: ³¹Si (³⁰Si(n,γ)³¹Si(β⁻)³¹P — ступенька s-процесса вверх по Z, Si→P) и ¹⁴C (петля ¹⁴N(n,p)¹⁴C(β⁻)¹⁴N). Дальнейшие β⁻-нестабильные изотопы добавятся вместе с target-ядрами выше ⁵⁶Ni.
  - 🟡 **(n,p)** — захват нейтрона с испусканием протона: A + n → A′ + p (Z→Z-1, A→A). Generic поле `neutronProtonResult` + правило `StarNeutronProtonReaction` **подключены**. Первый target — ¹⁴N(n,p)¹⁴C (космогенный ¹⁴C, радиоуглеродное датирование); ¹⁴C β⁻-нестабилен → вместе с `BetaMinusDecay` замыкает петлю ¹⁴N(n,p)¹⁴C(β⁻)¹⁴N. На ¹⁴N конкурирует с (n,γ)→¹⁵N — выбор канала случайный, оба наблюдаемы. Расширение — по мере появления target-ядер.
  - 🟡 **(n,α)** — захват нейтрона с испусканием α: A + n → A′ + ⁴He (Z→Z-2, A→A-3). Generic поле `neutronAlphaResult` + правило `StarNeutronAlphaReaction` **подключены**. Первый target — ¹⁷O(n,α)¹⁴C (кормит ту же радиоуглеродную петлю, что и (n,p); на ¹⁷O конкурирует с (n,γ)→¹⁸O, выбор случайный). Хрестоматийные ¹⁰B(n,α)⁷Li (B-NCT) и ⁶Li(n,α)³H ждут появления target-ядер ¹⁰B, ⁶Li (spallation/BBN).

- 🟡 **Радиоактивные распады.**
  - ✅ **β⁺-распад** — generic правило через `betaPlusDecayResult`. Все β⁺-нестабильные изотопы прописаны.
  - 🟡 **β⁻-распад** — нейтрон в ядре превращается в протон с испусканием электрона и антинейтрино: n → p + e⁻ + ν̄ₑ. Generic поле `betaMinusDecayResult` + правило `BetaMinusDecay` подключены (см. секцию (n,γ) выше). Targets — ³¹Si→³¹P и ¹⁴C→¹⁴N. Блокер s-процесса снят; расширение — по мере добавления нейтрон-избыточных изотопов.
  - 🟡 **Electron capture** — пока только хардкод ⁷Be+e⁻ в pp-II. Ждёт обобщения по образцу β⁺.
  - ⬜ **α-распад, γ-распад, спонтанное деление** — добавим, когда появятся соответствующие ядра (U/Th для α-распада, возбуждённые состояния для γ).

- ⬜ **Звезда как поглотитель + термическая ионизация.** Сейчас звезда только излучает. Поглощение нужно для замыкания цикла материи; термическая ионизация — чтобы рекомбинированные нейтральные атомы внутри звезды снова становились голыми ядрами.

- ⬜ **Нейтринные реакции.** На каждой звёздной реакции рождаются нейтрино, уносят энергию.
- ⬜ **Катализ через переходные состояния.** Активационный барьер в хим. реакциях.
- ⬜ **Гравитация.** Выключена. Без неё нет планетообразования и аккреции.
- 🟡 **Кулоновское отталкивание электронных оболочек** — упрощено до функции от числа электронов.

Дальше — пребиотика и жизнь: ферменты, автокатализ, репликация. Отдельная эпоха.

## Стратегические вопросы геймплея

1. **Что считать прогрессией?** Жизнь? Технологическое дерево? Колонизация? — это направит, какие механики достраивать.
2. **Эпохи симуляции.** Космогенез → нуклеосинтез → планеты → пребиотика → жизнь. На одной модели всю цепочку не покрыть; переходы между эпохами — игровая веха.
3. **«Реакторы» как игровая механика.** `SpaceModule`/`RecombinationModule` сжимают медленные естественные процессы — формула в духе Factorio, развивать.
